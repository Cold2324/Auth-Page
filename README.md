# Auth Page

## Installation

Clone this repo

```bash
git clone git@github.com:Cold2324/Auth-Page.git
cd Auth-Page
pnpm i
```

Start Server

```bash
pnpm run dev
```

Build Project

```bash
pnpm run build
```
